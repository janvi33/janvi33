## 🛠️ What I Do Best  
- **Architect**: Work with scalable, modular app architectures (**MVVM, MVI, Clean Architecture**) that hold up as features and teams grow  
- **Build**: Craft modern, expressive UIs with **Jetpack Compose** and ship cross-platform features that move fast without breaking  
- **Secure**: Implement **biometric authentication, certificate pinning, and data encryption** for high-trust, customer-facing experiences  
- **Optimize**: Profile and tune apps to cut startup time, shrink memory footprint, and crush crashes before customers ever see them  
- **Integrate**: Wire mobile to backend ecosystems via **GraphQL and REST**, with thoughtful caching, offline strategies, and error handling  
- **Collaborate**: Partner closely with product, design, and backend teams to turn ideas into features that actually ship  
- **Deliver Impact**: Turn business needs into mobile features that lift engagement, revenue, and customer trust
